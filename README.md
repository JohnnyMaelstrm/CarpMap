CarpMap

An interactive map of carp stocking locations in Finland, built to make it easier to find good fishing spots. Instead of digging through stocking registers manually, you can see all recorded stocking events on a map, filter by region or municipality, and check the stocking history of any lake with a single click.

What it shows:

All lakes stocked with carp in the Pirkanmaa and Varsinais-Suomi regions, 1980–2025
83 lakes, 219 stocking events in total
Per lake: stocking history (year, quantity, fish age and weight), municipality, coordinates, and a direct link to Google Maps

Marker size scales with total fish stocked — bigger dot means more fish over the years.
Data source
Stocking data comes from the Finnish ELY Centre (ELY-keskus) public stocking register. Lake coordinates are fetched from the Finnish Environment Institute (SYKE) official lake database using the unique lake ID (järvinumero), so locations are accurate rather than relying on name-based geocoding.
